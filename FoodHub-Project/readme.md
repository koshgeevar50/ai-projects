# Foodhub Project
This project showcases reading a csv file with various columns into a python dataframe. Then running general python functions on the dataframe and also various statistical functions on the dataframe.
Showcases using loc function to get different features based on the values within the feature and using groupby to group by various features and run other statistical functions on the results.
It also uses seaborn to do various plots like histograms, boxplots, pairplots and countplots for univariate, bivariate and multivariate analysis showcasing the dependancy between various features.
